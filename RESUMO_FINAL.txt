🎨 COMO CUSTOMIZAR:
═══════════════════════════════════════════════════════════════════════════════

MUDAR NOME DA BARBEARIA
   1. frontend/index.html (linha 7, título)
   2. frontend/index.html (linha 42, navbar)
   3. Dados aparecem automaticamente

MUDAR CORES
   1. Abra: frontend/css/style.css
   2. Edite: :root { --primary-color, --secondary-color, etc }
   3. Salve e atualize navegador

ADICIONAR FOTOS DE BARBEIROS
   1. Coloque em: frontend/assets/barbeiro1.jpg (2, 3)
   2. Tamanho recomendado: 200x200px ou maior
   3. Serão automaticamente carregadas

ADICIONAR NOVOS SERVIÇOS
   1. Opção A (recomendada - via API):
      curl -X POST http://localhost:3000/api/servicos \
        -H "Content-Type: application/json" \
        -d '{"nome":"Novo","preco":25,"tempo":40}'
   
   2. Opção B (direto no banco):
      Abra: backend/database/barbearia_sense.db
      Edite tabela "servicos"

MUDAR HORÁRIOS
   1. frontend/index.html (seção info)
   2. frontend/js/main.js (array de horários)

📧 CONFIGURAR EMAIL (Opcional):
═══════════════════════════════════════════════════════════════════════════════

1. COM GMAIL:
   a. Ative 2FA: https://myaccount.google.com
   b. Gere App Password: https://myaccount.google.com/apppasswords
   c. Edite backend/.env:
      EMAIL_HOST=smtp.gmail.com
      EMAIL_PORT=587
      EMAIL_USER=seu_email@gmail.com
      EMAIL_PASS=sua_app_password

2. COM MAILTRAP (desenvolvimento):
   a. Cadastre: https://mailtrap.io
   b. Copie credenciais
   c. Cole em .env

3. SEM EMAIL:
   Deixe .env com EMAIL_HOST vazio (padrão, tudo funciona)

🐛 PROBLEMAS COMUNS:
═══════════════════════════════════════════════════════════════════════════════

"Porta 3000 já em uso"
   → Mude a porta em .env: PORT=3001

"Erro ao instalar npm"
   → Execute: npm cache clean --force
   → Depois: npm install

"Frontend não carrega dados"
   → Verifique se backend está rodando
   → Abra console (F12) para ver erros

"Imagens não aparecem"
   → Coloque em: frontend/assets/
   → Verifique nomes: logo.png, barbeiro1.jpg, etc

"Agendamento não salva"
   → Verifique console (F12) para erros
   → Verifique terminal do backend
