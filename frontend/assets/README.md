# 📸 Guia de Imagens - Barbearia Sense

## Logotipo

O logotipo já foi preparado conforme solicitou. Ele deve ser colocado em:
```
frontend/assets/logo.png
```

**Características do logotipo fornecido:**
- Imagem professional com tesoura, barba e pincél
- Cores: Preto, Dourado, Branco
- Texto "Sense" com "BARBEARIA"
- Pronto para uso

**Como adicionar:**
1. Copie a imagem do logotipo que foi fornecida
2. Cole em `frontend/assets/logo.png`
3. Pronto! Será usado automaticamente

## Fotos dos Barbeiros

Adicione as fotos em:
- `frontend/assets/barbeiro1.jpg` - João Silva
- `frontend/assets/barbeiro2.jpg` - Carlos Santos
- `frontend/assets/barbeiro3.jpg` - Miguel Costa

**Especificações recomendadas:**
- Formato: JPG ou PNG
- Tamanho: Mínimo 200x200px
- Proporção: Quadrada (1:1)
- Qualidade: Alta (otimizada para web)

## Imagem Padrão

Se uma imagem não for encontrada, será usada:
- `frontend/assets/default-barbeiro.jpg`

Você pode adicionar uma imagem genérica também.

## Estrutura de Assets

```
frontend/assets/
├── logo.png              # Logotipo da barbearia (FORNECIDO)
├── barbeiro1.jpg         # Foto João Silva
├── barbeiro2.jpg         # Foto Carlos Santos
├── barbeiro3.jpg         # Foto Miguel Costa
└── default-barbeiro.jpg  # Imagem padrão (opcional)
```

## Dicas

1. **Otimize as imagens** para web (comprimir sem perder qualidade)
2. **Use formato JPG** para fotos (melhor compressão)
3. **Use formato PNG** para logotipos (transparência)
4. **Teste no navegador** para garantir qualidade

## Ferramentas Recomendadas

- **TinyPNG** (https://tinypng.com) - Comprimir imagens
- **Canva** (https://canva.com) - Criar designs
- **Photopea** (https://photopea.com) - Editor online
